클래스 탈퇴하기

꿈꾸는 5학년 1반 클래스 탈퇴 전 꼭 확인하세요.
클래스에서 탈퇴 시 작성한 글, 사진, 댓글 등을 삭제하거나 수정할 수 없습니다.
원하시면 탈퇴 전에 삭제 혹은 수정하시기 바랍니다.

취소 탈퇴하기

클래스에 멤버가 있을 시 운영자는 탈퇴가 불가능합니다.

운영자 위임 신청을 원하신다면 ‘클래스 설정 > 클래스 관리 > 운영자 위임 신청’을 통해 진행해 주시기 바랍니다.
